Junior Frontend Developer

Full-time Entry-level Remote Optional

We're looking for a talented Junior Frontend Developer to join our growing team. In this role, you'll work closely with senior developers and designers to build engaging user interfaces for our clients.

Key Responsibilities:

  • Develop responsive web interfaces using HTML, CSS, and JavaScript
  • Implement designs from Figma/Adobe XD wireframes
  • Work with modern frontend frameworks like React or Vue
  • Collaborate with the team for code reviews and problem-solving
  • Test and debug code to ensure cross-browser compatibility

Required Qualifications:

  • Degree in Computer Science, Web Development, or related field OR completion of a recognized coding bootcamp
  • Strong understanding of HTML5, CSS3, and modern JavaScript (ES6+)
  • Basic knowledge of at least one JavaScript framework (React preferred)
  • Understanding of responsive design principles
  • Familiarity with version control systems (Git)

Junior Backend Developer

Full-time Entry-level Remote Optional

We're seeking a Junior Backend Developer to help build robust server-side applications. This role is perfect for someone who enjoys working with databases and server architecture.

Key Responsibilities:

  • Develop and maintain server-side logic using Node.js, Python, or PHP
  • Design and implement database schemas and queries
  • Create and maintain RESTful APIs
  • Implement authentication and authorization mechanisms
  • Work with senior developers to optimize application performance

Required Qualifications:

  • Degree in Computer Science, Software Engineering, or related field OR completion of a recognized coding bootcamp
  • Knowledge of at least one backend language (Node.js, Python, PHP)
  • Basic understanding of database systems (SQL and NoSQL)
  • Familiarity with API design principles
  • Basic understanding of web security practices
